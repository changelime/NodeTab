# NodeTab
一个基于Node Garden的Chrome新标签页扩展

## 安装
Chrome 商店(暂无)

## 开发
```sh
    git clone https://github.com/changelime/NodeTab.git
    cd NodeTab
    npm run install
    //打开chrome扩展管理烈面，加载NodeTab目录
```

## 构建
```sh
    npm run built
    //打开chrome扩展管理烈面，加载NodeTab/dest目录
```

## 使用
扩展有3种状态
1. 时钟
1. 时钟和常用网站
1. 常用网站

更改状态
* 方向键上下，或者鼠标滚轮控制状态。
* 状态会保存（即使再新建一个标签）直到再次被更改
